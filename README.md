## :about_project: About project

Bitwallet is a simple way to organize your finance transactions and monitoring your finance health through the Dashboard with the resume transactions. 

## :feature_roadmap: Features and Roadmap

- [x] Transaction CRUD 
- [ ] User CRUD
- [ ] Category CRUD
- [ ] Payment CRUD


## :rocket: Technologies

This project was developed to help people to organize yours finances transactions using these following technologies

- [Bootstrap](https://getbootstrap.com/docs/4.6/getting-started/introduction/)
- [Javascript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [MySQL Database](https://www.mysql.com/)
- [Node.JS with Express](https://expressjs.com/)

## :getting_started: Getting stated

### Start API 

First, let's install the the dependency: 

```
    cd ./server
    npm install
```

Second, prepare environment and build the database

```
npm run-script build
```

Finally lets start the api service 

```
npm start
```

### Start Web project

Let's install the the dependency and build de project: 

```
    cd ../web
    npm install
    npm start
```
